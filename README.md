Batch Converter for Blender

Convert multiple 3D files in batch with a single click.

## Features
* Supported Formats: BLEND, FBX, OBJ, STL, glTF, GLB, ABC, PLY, DAE.
* Recursive folder processing.
* Preserves folder structure.
* Advanced mesh options (Join, Triangulate, Bake Transforms).

## Support the project
This tool is open-source and free to use.

## Installation
1. Download the latest release as a ZIP file.
2. In Blender, go to Edit > Preferences > Add-ons.
3. Click "Install" and select the ZIP file.
